# NatsuBot完整指令列表(v3.2)

### 【地区捷运】

::: info 公众指令﻿：

· `>点列表`：列出所有传送点﻿

· `>点 {地区编号}`：bot传送到指定公共地区的传送点（注意指令中间的空格）

:::

::: tip 白名单通用指令﻿：

· `>设点`：显示设点帮助信息。﻿

· `>设点 [权限] [名称] +[地点颜色]`：设置传送点。

详细说明：权限：a（所有人可用）或 b（仅限白名单成﻿员）。名称：不含空格的地区名称。地点颜色：可选（如 +FFA500，注意此处的加号）﻿示例：`>设点 a 传送点示例 +FFA500`

· `>删点 {地区编号}`：删除传送点（白名单成员仅可删自己设置的传送）点。

:::

::: warning 管理员指令：

· `>配置 [数量]`：调整白名单成员可设置的传送点总上限（默认3）。数量不能小于当前已存在的白名单点数量，且受总点数上限7限制。﻿

· `>加白 [真实ID] [显示ID]`：添加白名单成员。真实ID中可插入“分隔”以避免输入异常（Bot自动去除）。若玩家有双ID（显示名≠真实ID），需提供显示名，否则显示名默认为真实ID。﻿示例：`>加白 csy010m分隔d csy010**`

· `>去白 [真实ID]`：移除白名单成员。真实ID同样可带“分隔”。﻿

:::

其他说明﻿

· 白名单成员每人最多同时拥有1个传送点。﻿

· 地区编号码全局递增，删除后不重用。﻿

### 【挂机系统】

挂机系统搭载在双Bot上（ONEGAME 与NatsuCow\_），对应的指令前缀分别为“#”或“+”，比如：﻿

`#挂机`：调用ONEGAME；﻿

`+挂机`：调用NatsuCow\_。﻿

::: tip 白名单指令﻿

• `挂机`：发起挂机请求。

• `解锁`：解除Bot的锁定状态。仅限前一锁定者本人或管理员使用。

:::

::: warning 管理员指令﻿

· `加白 [真实ID] [异常状态显示ID]`：添加白名单成员。真实ID中可插入“分隔”字符以避免输入异常。若玩家有异常ID（聊天栏显示名≠真实ID），需提供显示名，否则显示名默认为真实ID。﻿

示例：`#加白 csy010m分隔d csy010\*\*`﻿

· `去白 [真实ID]`：移除白名单成员。真实ID同样可带“分隔”。﻿

· `白名单列表`：查看所有白名单成员列表。

:::﻿

其他说明﻿

· `/tpa ONEGAME/NatsuCow\_`：请求传送到Bot身边。锁定状态下仅允许锁定者本人和管理员，未锁定时仅白名单成员可用。﻿

· `/tpahere ONEGAME/NatsuCow\_`：请求Bot传送到你身边。锁定状态下所有人均被拒绝，未锁定时仅白名单成员可用。﻿

· 白名单【Net】与【Net+】共用。﻿

• 真人验证系统：ID异常的玩家在执行敏感操作时需要验证真实ID以验证是否为白名单成员（向bot私聊任意内容以验证`/msg ONEGAME或NatsuCow\_  KFCV50`）﻿

### 【小夏互联】

消息屏蔽前缀：“`>1`”，带有该前缀的消息会被Bot屏蔽，不会被互传至游戏内。﻿

::: warning 管理员指令：

· `#关机`：关闭互联﻿

· `#开机`：开启互联﻿

· `#重启`：重启互联﻿

· `#状态`：查看互联状态﻿

:::

补充：﻿

· 以上指令通过游戏内公屏聊天发送，Bot 自动识别并响应。﻿

· 如遇问题，请联系管理员获取帮助。﻿
